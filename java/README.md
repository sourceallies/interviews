# Java Project

## Getting Started

```bash
./gradlew build
```

## Tech Stack
- Java 21
- JUnit 5
- Spring Boot 3.2
    - Web
    - JPA
    - Lombok
- Gradle

## Useful Commands

| Command          | Description       |
| ---------------- | ----------------- |
| `./gradlew build`   | build the project |
| `./gradlew test`    |  run tests        |
| `./gradlew bootRun` | start the server  |
