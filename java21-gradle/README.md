# Java Project

## Getting Started

```bash [Run]
gradle build
```

## Tech Stack
- Java 21
- JUnit 5
- Spring Boot 3.2 
    - Web
    - JPA
    - Lombok
- Gradle

## Useful Commands

| Command          | Description       |
| ---------------- | ----------------- |
| `gradle build`   | build the project |
| `gradle test`    |  run tests        |
| `gradle bootRun` | start the server  |
